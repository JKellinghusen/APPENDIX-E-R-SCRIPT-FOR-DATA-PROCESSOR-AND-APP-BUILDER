To use the Workplace Wellbeing Assessment Tool:
1. Export the survey data from Qualtrics as a CSV and save it in the same directory as this file
2. Update line 3 in "R Data Processor" to utilize the newly exported CSV
3. Run "app.R" to run the dashboard within R
4. To see the app in a browser, close the dashboard in "app.R" and run "run_app.R" the navigate to http://localhost:1234/ in a browser
